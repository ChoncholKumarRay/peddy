Peddy - React Conversion Demo

React Components Structure

The original HTML/CSS/JavaScript application has been successfully converted to React using Vue.js components in JSX format.

Components Created:

  • Header.jsx - Navigation component
  • Banner.jsx - Hero section
  • MainSection.jsx - Main content area
  • CategoryButton.jsx - Category filters
  • PetCard.jsx - Individual pet cards
  • LikedPets.jsx - Liked pets sidebar
  • Spinner.jsx - Loading indicator
  • PetDetailsModal.jsx - Pet details modal
  • AdoptionModal.jsx - Adoption confirmation
  • Footer.jsx - Footer section

Features Implemented:

  • ✅ Category filtering
  • ✅ Pet sorting by price
  • ✅ Like/unlike pets
  • ✅ Pet details modal
  • ✅ Adoption process with countdown
  • ✅ Responsive design
  • ✅ Loading states
  • ✅ Error handling

Setup Instructions

Note: This is a React application that requires Node.js to run. Please install Node.js first.

1. Install Node.js: Download from nodejs.org

2. Install dependencies: npm install

3. Start development server: npm run dev

4. Open browser: Navigate to http://localhost:3000
